# Google_Maps_ScraperV0.1
 A Google maps scraper with Selenium, generating leads efficiently

 ### Get Started (Install)

Create a fresh virtual environnement :

* Mac OS :
   ```bash
   # Assuming the 2nd venv is your virtual environnement's name
   python3 -m venv venv
   ```

* Windows :
   ```bash
   # Assuming the 2nd venv is your virtual environnement's name
   python3 -m venv venv
   ```
